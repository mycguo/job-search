## ğŸ”§ Technical Documentation

### Architecture

- **Data Layer:** JSON storage (`storage/json_db.py`, `storage/interview_db.py`)
- **Models:** Application dataclasses (`models/application.py`, `models/interview_prep.py`)
- **AI Layer:** LangChain + Gemini (`ai/job_matcher.py`)
- **UI Layer:** Streamlit pages (`pages/`)
- **NL Processing:** Hybrid regex + LLM (`app.py`)
- **Analytics:** Plotly visualizations (`pages/dashboard.py`)
- **Interview Prep:** Hybrid JSON + Vector (`pages/interview_prep.py`)

### Key Technologies

- **Language:** Python 3.13
- **Framework:** Streamlit
- **AI:** Google Gemini 2.5 Flash (temperature=0.0)
- **Embeddings:** gemini-embedding-001
- **Visualizations:** Plotly Express & Graph Objects
- **Testing:** Pytest
- **Storage:** JSON (upgradeable)

### Performance Metrics

- **Application tracking:** < 1 second
- **AI analysis:** 5-8 seconds
- **Natural language:** 2-4 seconds
- **Dashboard load:** < 1 second
- **Chart rendering:** < 500ms
